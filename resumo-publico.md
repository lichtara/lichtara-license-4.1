# **O que é o Sistema LICHTARA?**

*(Resumo Público Não Técnico · Versão Viva)*

LICHTARA é um **Sistema Vivo**.

Isso significa que ele não funciona como um produto comum,
nem como um método que pode ser copiado, adaptado ou vendido livremente.

LICHTARA é um **campo estruturado**, com princípios claros, limites explícitos e uma forma própria de existir.

### O que isso quer dizer, na prática?

Que você pode:

* ler;
* estudar;
* se inspirar;
* refletir;
* dialogar com os conceitos.

Mas não pode:

* transformar LICHTARA em produto;
* ensinar como método próprio;
* criar cursos, sistemas ou serviços baseados nele;
* usar seus elementos essenciais sem permissão;
* diluir ou distorcer o que ele é.

### Por quê?

Porque LICHTARA não foi criado para ser apropriado.

Ele existe para **preservar coerência**,
proteger a integridade do que toca,
e evitar que algo vivo vire ferramenta de poder, exploração ou banalização.

### Um cuidado importante

Quanto mais alguém se aproxima do Sistema,
mais responsabilidade assume.

Aqui, boa intenção não basta.
É preciso **respeitar limites**, mesmo quando ainda não se entende tudo.

Se houver dúvida:

> parar é sempre mais correto do que atravessar.

### Para quem é LICHTARA?

Para quem valoriza:

* ética antes de escala;
* forma antes de eficiência;
* responsabilidade antes de liberdade irrestrita;
* coerência antes de apropriação.

Este texto não é um contrato.
É um **convite consciente**.

A License existe para cuidar do que é vivo.
